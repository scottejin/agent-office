# Retro Agent Ops Room

Playful local dashboard showing the main agent plus subagents as a retro operations office floor.

## Run locally

From this workspace:

```bash
cd /Users/crab/.openclaw/workspace/retro-ops-room
python3 -m http.server 4173
```

Then open:

- http://localhost:4173
